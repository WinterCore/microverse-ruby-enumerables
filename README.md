# Advanced Building Blocks - Enumerables

# Description

This is a ruby implementation of some of the methods of the [ruby enumerable module](https://ruby-doc.org/core-2.7.1/Enumerable.html).

It includes the following
- `my_each`
- `my_each_with_index`
- `my_select`
- `my_all`
- `my_any`
- `my_none`
- `my_count`
- `my_map`
- `my_inject`
